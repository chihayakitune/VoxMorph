# VoxMorph 高機能化プラン — 「リアルで艶のある女声」への変換パラメータ設計

方針: エフェクト付与ではなく、**声の生成モデル(音源=声帯 × フィルタ=声道)の各要素を独立に操作する**ことで自然さを上げる。
男声→女声が不自然になる原因は主に4つあり、それぞれに対応するパラメータ群を用意する。

| 不自然さの原因 | 対応パラメータ群 |
|---|---|
| フォルマントの一律シフト(実際の男女差は非一様) | A. 個別フォルマント |
| 声帯音源の質の違いが再現されない(女声は息漏れが多く柔らかい) | B. 音源モデリング |
| ピッチの「動き」が男声のまま(女声は抑揚が大きい) | C. ピッチ挙動 |
| 変換アーティファクト(ゆらぎ・こもり・子音の破綻) | D. 品質改善 |

## A. 個別フォルマント操作(F1/F2/F3)

音声学的背景: F1=顎の開き・喉の広さ、F2=舌の前後位置、F3=唇・声道前部の長さに対応する。
実際の男女差は一律ではなく、**F1は小さめ(+10〜15%)、F2は大きめ(+15〜20%)、F3は中間(+10〜15%)**が典型。
現在の一律シフトで上げすぎると「チップマンク化」するのは、F1が上がりすぎるため。F2を優先的に上げると「若く明るい」方向に動く。

- **F1 / F2 / F3 Shift (±12st)** — 全体Formantノブへの加算として動作
- **F1 / F2 / F3 Gain (±12dB)** — 各フォルマント帯域の強さ。F3付近の持ち上げは「艶・張り」に直結
- **Envelope Detail** — LPC次数(包絡の細かさ)。上げると正確、上げすぎると揺らぐ
- **Formant Smooth** — フォルマント軌跡の時間平滑。誤追跡時のワブル防止

実装: グレイン単位でLPC(12kHz換算・次数14〜18)→ 極からF1〜F3抽出+連続性トラッキング → スペクトル包絡を区分単調ワーピング+帯域ゲイン → 残差×新包絡で再合成。無声区間はバイパスしクロスフェード。

## B. 音源(声帯)モデリング — 「艶」の本体

女声の声帯振動は開放時間が長く(高OQ)、(1)基音が強く、(2)高域の減衰が急で、(3)息成分(aspiration)が多い。ここを再現しないと「ピッチとフォルマントは女性なのに声帯が男性」という違和感が残る。

- **Breath (0〜100%)** — フォルマント包絡で整形した息ノイズを**ピッチ同期で声門開放相に**混ぜる(単なるノイズ付加ではなく声門音源の再現。これが「息っぽい艶」)
- **Softness / Tilt (±dB/oct)** — スペクトル傾斜。+方向で柔らかく囁き寄り、−方向で張りのある声
- **Fullness (H1 Gain)** — 基音付近の強さ。細くなりすぎた声に「体」を戻す
- **Grain Shape** — グレイン窓形状で声門パルスの鋭さを近似操作(鋭=バリバリした声、鈍=丸い声)

## C. ピッチ挙動 — 「女声らしさはピッチの動きに宿る」

同じ+7stでも、抑揚が男声のまま平坦だと一気に不自然になる。逆にここを整えると聴感が跳ね上がる。

- **Pitch Range (50〜200%)** — 検出f0の抑揚をPitch Centerを支点に拡大/圧縮。女声化では110〜140%が目安
- **Pitch Center (Hz)** — 抑揚スケールの支点となる目標ピッチ
- **Natural Jitter (0〜100%)** — ごく微小なピッチ揺らぎを付加。機械っぽさの緩和(特にRobotize/強変換時)
- **Octave Guard** — ピッチ検出のオクターブ誤り抑制(履歴ベース)。「ウォブル」「一瞬野太くなる」問題の根治

## D. 変換品質の改善(ボイチェン特有の問題)

- **子音独立シフト (Consonant Shift)** — 無声子音(/s/ /sh/等)だけ別係数でシフト。女声の/s/はスペクトル重心が高く、ここが男声のままだと老けて聞こえる。上げすぎるとリスプ化するので独立ノブが必須
- **有声/無声境界のクロスフェード改良** — 語頭・語尾のプチノイズ対策
- **高フォルマント時のバズ低減** — グレイン幅の適応制御(現在は固定比)
- **低遅延モード** — 現在43ms→目標20ms以下(最低f0の設定と検出窓の見直し)。配信・実況用途に必須

## 推奨実装順

| フェーズ | 内容 | 効果/工数 |
|---|---|---|
| 1 | C: Pitch Range/Center/Octave Guard + D: 子音独立シフト + B: Breath/Tilt | **効果大・工数小**(現エンジンに直接足せる) |
| 2 | A: 個別フォルマント一式(LPC包絡ワーピング) | 効果大・工数中(本機の切り札) |
| 3 | B残り(Grain Shape/H1) + D残り(低遅延・境界改良) | 仕上げ |

フェーズ1だけでも「ピッチ・抑揚・息・子音」が揃い、女声化の質感は大きく変わる見込み。
